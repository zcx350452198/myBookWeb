旧版本安装步骤

第一步：使用 SSH 连接工具，如 PuTTY、XShell、SecureCRT 等，连接到您的 Linux 服务器。

相应命令如下：
# wget http://vpsmate.arsh.cn/tools/install.py

第二步：下载特定版本的VPSMate安装包，并改名为vpsmate.tar.gz，将安装包与安装脚本放于同一目录下。

相应命令如下：
wget http://vpsmate.arsh.cn/Uploads/releases/vpsmate-1.0b9.tar.gz
mv vpsmate-1.0b9.tar.gz vpsmate.tar.gz

第三步：在命令行中运行：python install.py 即可完成此版本的安装。

相应命令如下：
# python install.py
